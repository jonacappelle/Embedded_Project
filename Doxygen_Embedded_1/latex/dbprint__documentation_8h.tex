\hypertarget{dbprint__documentation_8h_ENABLE}{}\doxysubsection{Enable/disable dbprint using definition in $<$tt$>$debug\+\_\+dbprint.\+h$<$/tt$>$}\label{dbprint__documentation_8h_ENABLE}
In the file {\ttfamily \mbox{\hyperlink{debug__dbprint_8h}{debug\+\_\+dbprint.\+h}}} {\bfseries{dbprint U\+A\+RT functionality can be enabled/disabled}} with the definition {\ttfamily \#define D\+E\+B\+U\+G\+\_\+\+D\+B\+P\+R\+I\+NT}. \hypertarget{dbprint__documentation_8h_DBPRINT}{}\doxysubsection{More info about dbprint and V\+C\+OM}\label{dbprint__documentation_8h_DBPRINT}
V\+C\+OM is an on-\/board (S\+L\+S\+T\+K3400A) U\+A\+RT to U\+SB converter alongside the Segger J-\/\+Link debugger, connected with microcontroller pins {\ttfamily P\+A0} (RX) and {\ttfamily P\+F2} (TX). This converter can then be used with Putty or another serial port program.

\begin{DoxyNote}{Note}
When you want to debug using V\+C\+OM with interrupt functionality disabled, you can use the following initialization settings\+:~\newline
 {\ttfamily dbprint\+\_\+\+I\+N\+I\+T(\+U\+S\+A\+R\+T1, 4, true, false);}

When using {\ttfamily dbprint} functionality, the following settings are used\+:~\newline

\begin{DoxyItemize}
\item Baudrate = 115200
\item 8 databits
\item 1 stopbit
\item No parity
\end{DoxyItemize}
\end{DoxyNote}
